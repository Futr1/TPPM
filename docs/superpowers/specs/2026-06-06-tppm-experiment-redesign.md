# TPPM 实验结构重设计方案

**日期**：2026-06-06
**状态**：待实施
**范围**：仅修改实验部分（第4-8节），不修改方法部分（第3节）

---

## 1. 设计目标

基于现有三个数据集（PsyDial、LoCoMo、PersonaMem），围绕 TPPM 的核心创新点重新设计实验结构，使每个创新点都有明确的验证路径。

**核心变化**：
- 移除 TalkLife MoC（实验二 Layer 2）
- 不引入新数据集
- 通过消融实验升级和分析子节升级来验证动态机制（衰减/固化）
- 场景条件化分支不单独验证，保留为常规消融项
- 混合式设计由扰码/噪声记忆对照覆盖，不需要额外实验

## 2. TPPM 核心创新点与验证路径映射

| 创新点 | 核心机制 | 验证路径 |
|--------|---------|---------|
| ①心理画像中心记忆 | PPMU 7维槽位，只保留心理相关信息 | PersonaMem Acc + Unsupported Rate + 消融（w/o Structured Slots） |
| ②时间演化-固化 | 状态→特质固化（公式11，K次观测+Π>δ） | K敏感性扫描 + Π-score vs.简单计数对比 + 消融(w/o Consolidation) + 高不稳定压力测试 |
| ③时间演化-衰减 | 类型条件衰减 λ_affect > λ_stressor > λ_trait（公式13） | λ敏感性扫描 + 4种衰减策略对比（含随机差分） + 消融(w/o Decay) + 会话数敏感性 |
| ④场景条件化分支 | 同一用户不同情境的差异化表达（公式3） | δ_ctx敏感性 + 分支策略对比 + 消融项（w/o Branching） |
| ⑤三层记忆架构 | 工作/瞬时/长期三层 | LoCoMo 跨session QA + 会话数敏感性 |
| ⑥混合式设计 | LLM抽取 + 规则驱动演化 | 扰码/噪声记忆对照（已覆盖） |
| ⑦画像感知检索 | 5维检索分数（公式8） | PsyDial OA + LoCoMo QA + 消融（w/o Retrieval） |

## 4. 各实验详细设计

### 4.1 实验一：基本不变

与现版完全一致：
- Layer 1：PsyDial 101段测试对话，8维咨询技能 + OA，GPT-4o 评分
- Layer 2：LoCoMo QA（5类F1）+ Event Summarization（FactScore + ROUGE-L）
- 两层综合解读保留

**唯一微调**：综合解读段落中，将"心理领域特异性由实验二 Layer 2（TalkLife MoC）独立验证"改为"动态记忆机制的设计最优性由实验二 Layer 2 验证，各机制的不可替代性由实验三消融验证"。

### 4.2 实验二：重构

#### 4.2.1 标题与定位

**标题**：实验二：心理画像记忆建模与动态机制验证

**实验目的**：实验一的双层评估验证了 TPPM 在回复生成中的效果及其与通用记忆能力的关联。实验二进一步验证：(1) 系统是否准确抽取了用户画像槽位（Layer 1·PersonaMem）；(2) TPPM 的三个动态记忆机制——状态→特质固化、类型条件衰减、场景条件分支——的设计选择是否最优（Layer 2·参数敏感性分析 + 替代设计对比）。

#### 4.2.2 Layer 1：PersonaMem（不变）

与现版完全一致。包括：
- 评估协议、设计优势、局限性说明
- Table: PersonaMem 画像槽位抽取评估（6列）
- 论证逻辑

#### 4.2.3 Layer 2：动态机制参数敏感性分析（新）

**设计动机**。Layer 1 验证了画像槽位的静态抽取质量，但 TPPM 的核心创新在于动态记忆管理——时间衰减、状态→特质固化、场景条件分支。Layer 2 通过参数敏感性分析回答"每个动态机制取什么参数值最有效"，并对比替代设计，验证 TPPM 的具体设计选择是否优于简单替代方案。

**与实验三消融的区别**。实验三回答"每个机制**是否不可替代**"（消融：移除后性能是否退化），Layer 2 回答"每个机制**如何最优**"（敏感性：参数取什么值最好，TPPM的设计选择是否比替代方案更优）。两者完全互补，无冗余。

##### 2a. 固化机制参数敏感性分析

**扫描参数**：固化所需最少会话数 K ∈ {1, 2, 3, 5, 7, 10}

**报告指标**：
- PersonaMem Q4（偏好演化追踪）随K的变化
- Trait Consolidation Precision / Recall（而非合并F1，以避免掩盖Precision-Recall权衡）
- LoCoMo QA Multi-hop F1 随K的变化

**替代设计对比**（回答"TPPM的Π-score固化是否优于简单规则"）：
1. **TPPM 固化**（公式11）：Π_i(t) > δ_promote ∧ N_session ≥ K
2. **简单计数固化**：N_session ≥ K（无Π-score门控，纯计数）
3. **固定窗口固化**：最近K个session中至少出现K_min次

在相同K值下比较三种固化策略，报告Q4和Trait Consolidation Precision/Recall。

**论证逻辑**：
- 若存在最优K值且呈现清晰的Precision-Recall权衡（K小→Precision低Recall高，K大→Precision高Recall低），则固化阈值的设计是必要的
- 若TPPM Π-score固化优于简单计数固化，则Π-score门控不仅"有用"而且"最优"
- 若简单计数固化与TPPM相当，则需诚实报告并在讨论中分析原因

##### 2b. 衰减机制参数敏感性分析

**扫描参数**：衰减率尺度 λ_scale ∈ {0.1, 0.3, 0.5, 1.0, 2.0}（乘以公式13的基础λ值）

**报告指标**：
- LoCoMo QA Temporal F1 随λ_scale的变化
- PersonaMem Q3（识别最新偏好）随λ_scale的变化
- 各维度槽位的Per-Slot F1随λ_scale的变化（诊断哪些维度对衰减率最敏感）

**衰减策略对比**（回答"差分衰减是否优于替代方案"）：
1. 无衰减（λ_d ≡ 0）
2. 统一衰减（单一λ，所有维度相同）
3. 随机差分衰减（随机排序的λ，维度间有差异但非TPPM的心理学排序）
4. TPPM差分衰减（λ_affect > λ_stressor > λ_coping ≈ λ_support > λ_trait）

**论证逻辑**：
- 若TPPM差分衰减 > 随机差分衰减 > 统一衰减 > 无衰减，则证明"不仅需要衰减，还需要按心理学排序的差分衰减"——这是最强论证
- 若TPPM差分衰减 ≈ 随机差分衰减 > 统一衰减，则仅证明"需要差分但不需特定排序"——需在讨论中承认
- 若统一衰减 ≈ 差分衰减 > 无衰减，则仅证明"需要衰减但不需要差分"——更弱但仍有价值

##### 2c. 场景分支机制参数敏感性分析

**扫描参数**：场景分支阈值 δ_ctx ∈ {0.3, 0.5, 0.7, 0.9}（公式3中的上下文相似度阈值，控制何时建立分支vs.融合）

**报告指标**：
- PersonaMem Q7（跨场景泛化）随δ_ctx的变化
- LoCoMo QA Adversarial F1 随δ_ctx的变化
- 分支数量随δ_ctx的变化（诊断分支粒度）

**分支策略对比**：
1. 无分支（全局画像更新，δ_ctx → ∞）
2. 二值分支（仅区分"危机"vs."非危机"两种场景）
3. TPPM连续分支（公式3，基于上下文相似度的软分支）

**论证逻辑**：
- 若TPPM连续分支 > 二值分支 > 无分支，则场景分支有效且细粒度分支优于粗粒度
- 若存在最优δ_ctx，则分支阈值的设计是必要的
- 若二值分支 ≈ TPPM连续分支，则说明当前数据集的场景多样性不足以区分细粒度分支和粗粒度分支，需在讨论中说明

#### 4.2.4 Layer 2 论证逻辑总结

Layer 2 的三个参数敏感性分析回答"每个动态机制如何最优"：

```
固化敏感性 → K最优值存在 + Π-score优于简单计数 → 固化设计不仅"有用"而且"最优"
衰减敏感性 → λ最优值存在 + 心理学排序差分优于随机差分 → 衰减排序有理论依据
分支敏感性 → δ_ctx最优值存在 + 连续分支优于二值分支 → 分支粒度有实际意义
```

与实验三消融（"是否不可替代"）互补，形成完整论证：
- 实验二 Layer 1：画像更准
- 实验二 Layer 2：每个机制的设计选择是最优的（敏感性分析 + 替代设计对比）
- 实验三：每个机制是不可替代的（消融）

## 7. 风险与缓解

| 风险 | 影响 | 缓解 |
|------|------|------|
| 无独立数据集验证动态机制 | 审稿人可能质疑"只有消融和敏感性不够" | 敏感性分析（设计最优性）+ 消融（不可替代性）+ 压力测试（失败边界）三线收敛；且实验在三个不同数据集上交叉验证 |
| 心理健康纵向特异性验证缺失 | 审稿人可能质疑"三个数据集都不是心理健康纵向数据" | 讨论中显式承认局限性；压力测试加强为心理健康场景；PersonaMem therapy子集深度分析 |
| PersonaMem子指标统计功效不足 | therapy子集(~9.1%)样本量小，消融/敏感性退化可能不显著 | 报告效应量(Cohen's d)而非仅p值；如子指标不显著则退而报告Overall Accuracy；必要时在全量15话题上补充 |
